To install gromit-mpx: "sudo apt-get install gromit-mpx"

Use provided Config file into /etc/gromit-mpx

1. How to start gromit-mpx ?
A. Run following command in Terminal:
    gromit-mpx --key "F9"

2. Trigger gromit-mpx or Toggle cursor to marker: F9 key

3. Clear/Erase Screen: SHIFT + F9

4. Toggle Marked Display: CRTL + F9

5. Quit gromit-mpx: ALT + F9

6. Pen (Orange Color): Tip of Pen

7. Pen (Green Color): Hold SHIFT

8. Pen (Red Color): Hold ALT

9. Eraser: Back of Pen

10. Highlighter (Cyan Color): Hold CRTL

11. Undo Highlight (Orange Color): Hold SHIFT + CRTL
